# Marchen

Marchen 是一个基于 Next.js 构建的现代化博客平台，集成 AI 辅助写作功能，支持文章摘要和标题自动生成、智能分类与标签推荐等

> **注意：** 项目目前处于开发阶段，功能可能不完整或存在变动。
